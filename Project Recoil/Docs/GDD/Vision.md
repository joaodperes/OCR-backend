# Vision

Project Recoil is a fast-paced, stylized arena FPS inspired by Offensive Combat.

## Design Pillars
1. Fun First
2. Respect the Player
3. Offline First
4. Easy to Learn, Hard to Master

## Primary Goals
- Fast arcade gunplay
- Team Deathmatch
- Persistent progression
- Competent offline bots
- Dedicated server support
- Steam release

## Non-Goals
- Battle Royale
- Mandatory online
- Hero shooter
- Live-service monetization
